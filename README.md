# 🚦 Análise Exploratória de Acidentes de Trânsito em Porto Alegre

Projeto desenvolvido para a disciplina de **Fundamentos de Ciência de Dados**, utilizando dados públicos de acidentes de trânsito de Porto Alegre para identificar padrões e gerar insights por meio de Análise Exploratória de Dados (EDA).

## 📊 Objetivos

O projeto busca responder às seguintes perguntas:

- Quais tipos de acidente geram mais vítimas graves ou fatais?
- Acidentes noturnos são mais graves do que os diurnos?
- Em quais dias da semana e horários ocorrem mais acidentes?

## 🛠️ Tecnologias

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📈 Análises realizadas

- Limpeza e tratamento dos dados
- Tratamento de valores ausentes
- Criação de visualizações
- Identificação de padrões temporais
- Comparação entre acidentes diurnos e noturnos
- Análise dos tipos de acidentes mais graves

## 📂 Base de Dados

Os dados utilizados são públicos e foram obtidos através do Portal de Dados Abertos de Porto Alegre.

[https://dadosabertos.poa.br/dataset/acidentes-de-transito-acidentes](https://dadosabertos.poa.br/dataset/acidentes-de-transito-acidentes)
