🚦 Análise Exploratória de Acidentes de Trânsito em Porto Alegre

Projeto desenvolvido para a disciplina de Fundamentos de Ciência de Dados utilizando a base pública de acidentes de trânsito de Porto Alegre.

🔍 O trabalho busca responder:

• Quais tipos de acidente geram mais vítimas graves ou fatais?

• Acidentes noturnos são mais graves que os diurnos?

• Em quais dias da semana e horários ocorrem mais acidentes?

🛠️ Tecnologias utilizadas:

• Python

• Pandas

• Matplotlib

• Jupyter Notebook

📊 O projeto inclui tratamento de dados e visualizações estatísticas
